# 기타 명령어

- commit
    
    
    $ git commit .
    

- push


    $ git push origin test
    

- 강제 pull


    $ git fetch --all
    $ git reset --hard origin/master

- .gitignore 엉켰을 때


    $ git rm --cached .idea -rf



- git push 이미 해버렸는데 흔적없이 되돌리고 싶을 때


    $ git checkout {{branch_name}}
    $ git reset --hard {{commit_id}}
    $ git push --force origin {{commit_id}}:{{branch_name}}
